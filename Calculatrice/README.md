# Calculatrice Web

# Description

Application web permettant d'effectuer des operations arithmetiques de base (addition, soustraction, multiplication, division) via une interface simple et intuitive.

# Technologies utilisees
- HTML
- CSS
- JavaScript

# Fonctionnalites
- Interface responsive
- Calculs e temps reel
- Gestion des erreurs (division par zero)

# Objectif

Ce projet m'a permis de renforcer ma maitrise du DOM, des evenements JavaScript et de la logique de programmation cote client.
